# Estratégia de Desenvolvimento - Lyra OS

1. **Configuração da Base:** Definir pacotes essenciais via `apt`.
2. **Branding:** Aplicar temas GNOME, papéis de parede e logotipos.
3. **Automação:** Criar scripts para configuração pós-instalação.
4. **Instalador:** Otimizar o Ubiquity ou utilizar a base do instalador do Ubuntu.
