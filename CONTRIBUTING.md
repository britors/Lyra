# Contributing to Lyra OS

Contributions are welcome! Whether it's bug reports, feature requests, or code contributions, please follow these guidelines:

1. **Fork the repository.**
2. **Create a new branch** for your feature or fix.
3. **Submit a pull request** with a clear description of your changes.
4. **Follow the code style** and conventions used in the project.

By contributing, you agree that your contributions will be licensed under the project's GPLv3 license.
